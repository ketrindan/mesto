# Проект: Место

## ***by Katya Danilenko***

Сервис *Mesto* - интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

### **Технологии**
* Flex-box верстка
* Grid Layout
* Адаптивная верстка
* Организация по БЭМ
* JavaScript
